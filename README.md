# 📝 Task Management API

A simple RESTful Task Management API built with Spring Boot.

---

## 🚀 Features

- Create, retrieve tasks
- Input validation
- Exception handling
- In-memory H2 database
- RESTful endpoints

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot 3
- Spring Data JPA
- H2 Database
- Maven
